kintone Connector for talend

kintone Connector for talend kc4tは Talend Open Studio 用のkintone接続コンポーネントです。

Talendコンポーネント

  • tKintoneInput | kintoneからJDBC接続可能なRDBMSへのレコードの読み込み
  • tKintoneOutput | RDBMS等からkintoneへレコードの書き込み

必要な環境

  • kintone SDK for Java version 0.6以降 https://github.com/kintone/java-sdk
  • JDK 7以降

ダウンロード

  • ソースコード/バイナリを含むダウンロードはGitHubリポジトリからお願いします https://github.com/r3it/kc4t

コンポーネントのインストール方法

  • Talend Open Studioを起動します
  • “カスタムコンポーネント” フォルダを設定します
    • 例として”Custom Components”という名前のフォルダを、ハードディスクのどこかに作成します
    • Studioの設定画面で Window-> Preferences-> Talend-> Components と遷移します
    • “User component folder” にいま作成した”Custom Components”フォルダを設定します
  • ダウンロードしたソースの中のsrcフォルダに有る tKintoneInput および tKintoneOutput フォルダを、あなたの “Custom Components” フォルダにコピーします
  • Component Designerパースペクティブで、それぞれのコンポーネントフォルダを右クリックして、「コンポーネントをパレットにプッシュ」を実行してください。
