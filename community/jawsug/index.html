JAWS-UGへの貢献

JAWS-UG（じょうずゆーじー）とは、日本全国に50以上の支部を持つ Amazon Web Services  のユーザーグループです。完全コミュニティ手動で活動しており、全国の各支部では、 AWS に関する様々な技術勉強会や人材交流が、毎週のように行われています。

Contributions

弊社の金春（こんぱる）がJAWS-UG全国代表、池上がJAWS-UG関西女子会のコアメンバー、西島がJAWS-UG沖縄のコアメンバーとして活動しており、その活動を積極的に支援しています。

また、GitHub上のJAWS-UGリポジトリへのコンテンツコントリビュートも積極的に行っています。

アールスリーインスティテュートとしては、コミュニティ主催のイベントへのスポンサードや、各種イベントへの社員の登壇、イベント運営を支援しています。
