2016年1月のコミュニティイベントのご案内

Posted on 2016-01-07 16:38

Koichiro Nishijima

無事2016年1月2日に海開きを済ませた西島です、こんにちは。

みなさま、明けましておめでとうございます。

今年もアールスリーインスティテュートの Cloud Developers Blog をよろしくお願い致します。

2016年も、弊社社員はあちこちのコミュニティイベントの主催や登壇、大規模イベント、カンファレンスなどなどにお邪魔いたしますので、よろしくお願い致します！

さて今日は、今月開催のコミュニティイベント＋αを幾つかお知らせいたします！

2016年1月9日（土）/ JAWS-UG 関西女子会 2016年新年会

https://jawsugkgirls.doorkeeper.jp/events/35309

JAWS-UG 関西女子会は 20代～40代まで 幅広い年齢層がいることに加え、業種も営業さんからデザイナー、インフラエンジニアと様々な方々がいらっしゃる、とてもゆるい集まりです！ 各自ご自由に食べ物・飲み物を持ち込んでの新年会！是非ご参加くださいｲｪｲL(‘ω’)┘三└(‘ω’)｣ｲｪｲ

2016年1月16日（土）/ kintone Café 関西女子会 Vol.1

https://kintonecafe-osaka.doorkeeper.jp/events/36096

kintone Caféに関西女子会が出来ました！ 初開催となるVol.1は、「kintoneを触ってみよう！」です。お子様連れでも大丈夫ですので、お気軽にご参加ください！

そしてこの2つのイベント、弊社新オフィスでの開催となりました〜！

そんなに広くない場所ですが、コミュニティに場所をご提供できること、とても嬉しく思います！無線LAN完備＆飲食可＆新しい椅子と机！皆様ゆったりのんびりご活用くださいませ。

（西島はまだ出社したことがないので、実は分からないのですが(^_^;) みなさまお先に体験してきてください）

2016年1月16日（土）/ JAWS-UG沖縄 ゼロから始めるモバイルアプリ開発 / AWS Mobile Hubハンズオン

https://jaws-ug-okinawa.doorkeeper.jp/events/36263

同日16日土曜日は、JAWS-UG沖縄にて【ゼロから始めるモバイルアプリ開発 / AWS Mobile Hubハンズオン】が開催されます！

アプリケーションのコードを全く書くこと無く、モバイルアプリが出来てしまうという強力なサービスであるAWS Mobile Hubのハンズオン！ 既に県外からも数名の方が「遊びに行くよ〜！」と宣言してくださっております。まだもう少し空きがありますので、是非ご参加ください！ 現時点で100ページ以上の大作資料になっていますが、まだ全部作りきっておりません。頑張ります・・・(^^;

今年の沖縄は異常に暑い＆雨が一段と多いので、少しはいい天気になってくれることを期待しましょう・・・。

1月以降のイベント

2月のとあるイベントに、弊社社員がお邪魔します！

しかし残念、まだ秘密ということなので、こちらは公開許可が出次第、またお知らせいたします。

JAWS DAYS 2016 サポートします！

http://jawsdays2016.jaws-ug.jp/

今年も3月12日に JAWS DAYS 2016 が開催されます。JAWS-UGが主催する最大のお祭りのひとつです。

弊社社員が運営ボランティアスタッフやセッションに登壇したりと、裏も表も一生懸命活動しております。

既に先行参加受付も始まっていますので、皆様是非3月の東京新宿でお会いしましょう！

弊社は今年も企業サポーターとしてJAWS-UGを支援していきます！よろしくお願い致します。

以上、今年もよろしくお願いいたします！
