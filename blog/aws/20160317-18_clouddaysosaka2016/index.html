Cloud Days 大阪 2016 出展レポート

Posted on 2016-03-23

Ikegami MIdori

皆さん3連休はゆっくり過ごされましたか！？

3月も半分が過ぎ、連日かなり日中暖かい日が続いていますね└(^o^└　)┘

朝夕との気温差が激しいので、体調にはくれぐれもお気を付け下さい！

わたしも寒暖差なのか、はたまた花粉症なのか、鼻炎にかなり苦しんでおります。。。

さてさて！先週末3月17日(木)・18日(金)の2日間に渡って開催されたCloud Days 大阪 2016の出展レポートをお届けいたしますよ！

今回はAWSさんのパートナーブースでの出展となりました！

Cloud Days って？

日経BP社主催の大規模なITイベントです。今年2016年はクラウド、ビッグデータ、モバイル＆ウエアラブル、セキュリティ＆ガバナンスそしてIoTにフォーカスし、東京・大阪、九州・名古屋・札幌の全国5ヶ所で開催されます。

Cloud Days 東京では、サイボウズさんのパートナーブースで出展させて頂きました！

その模様もブログでupしておりますので、こちらも是非ご覧になって下さいね(`･⊝･´)

Cloud Days 大阪でも・・・

会場の真ん中のスペースを陣取って、東京と同じくテスラがお出迎え！神々しいです…！

イベントの様子

参加されたお客様は、皆イベントに興味津々(p*･ω･)p

大阪のイベント出展は、AWS Cloud Roadshow 2015以来だったのですが、参加者の皆さんの熱量が凄かったです！

場所もグランフロントという立地の良さがあったためか、お昼休みや仕事帰りに立ち寄られる方も多くいらっしゃったような印象です。

AWSゾーンも大盛況！

スカイアーチさんのブースで顔抜き写真撮影をしてFacebookでupするとサバ缶がGETできちゃうのですよー！

このサバ缶ほんとにおいしいと大評判なのです！わたしもしっかりGETしましたε( 　ˇωˇ　 )эﾌｸﾞ～～

弊社も『IoTお試しセットあります！』と看板を掲げてデモをブースでご用意していたのですが、IoTという言葉に惹かれて「お試しセットってどんなものですか？」とブースにお立ち寄り頂いた方もたくさんいらっしゃいましたよ！

今回から期待の新人浅賀もイベントデビューでした！

さすがイベント慣れしている新人なだけあって、大活躍(っ´ω`c)♡

プレゼンも大盛況！

AWSさんのブース内のプレゼンステージでは金春が『AWSとkintoneを使った新しいSIの姿』というタイトルでハイスピードSIの事例紹介等をさせて頂きました！

1日目はおかげ様でボーナスセッションを獲得！

しかし他のパートナーさんのセッションも魅力的なセッションばかりでほんとに僅差でした！

AWSJのSAさんのお話が聞けるのも貴重かも！？

あまり外でお話を聞ける機会ってなかなかない気がします(っ´ω`c)♡

また空いた時間では、サイボウズさんのブースでも飛び入りで登壇させて頂きました！

こんなところにまで！？

イベントの合間には、他のブースも見て回っていたのですが、金春はこんなところにも出没していました！

わたしもクリアファイルもらってご満悦です！

全体として

大阪もだいぶクラウドの波が来ている予感！？

実状として、オンプレのサーバーを運用している人もまだまだ多いですが、クラウドへの関心の高さが窺えた2日間でした！

今後もいろいろなイベントに出展する予定がありますので、皆さん是非ブースに来て下さいね！

お待ちしています(๑•̀ㅂ•́)و✧
