AWS Cloud Roadshow 2015 大阪に出展しました！

Posted on 2015-12-03 15:30

Ikegami Midori

2015年12月2日水曜日（昨日ですよ昨日！）AWS Cloud Roadshow 2015 大阪にスポンサーとして出展してまいりました！

AWS Cloud Roadshow 2015 大阪 に出展！！

去年のRoadshowは 会社が違ったので 夜のJAWS-UG大阪のLT大会しか参加出来ず。関西でのパイ姫デビューを飾った記念すべき日だったのですが、今年はR3にJOINして初のスポンサーとしての出展でテンションも上がりまくりでした∠( ‘ω’)／ｷﾘｯ

スポンサーさんも豪華です！弊社のロゴも上のほうに並んでいてﾆﾖﾆﾖしてしまいます…！（そこか

ブースも受付を入ってすぐの場所で、スペースも広々！

看板の代わりに大きなスクリーンを使ってプロジェクタで投影して弊社サービスを紹介するという初の試みを行ったのですが、通りすがりの参加者の人たちの中には「このプロジェクタどこのですか？型番教えて下さい」と寄ってくる方もいらっしゃいました。

いろんな意味で目立っていたようです（いろんな意味で

「ほむらとサービス関係あるんですか？」と聞かれたり、「時間止められるサービスなんですか？」（ほむの能力）と聞かれたり。。。

もちろんた　だ　の　趣　味　

参加者の皆さんの関心事

今回はOpenBlock + SORACOM + AWS IoTを使ったデモもお見せしたのですが、わりと参加者の皆さん興味を持たれてました！

SORACOMとIoTはやはり関西でも関心度が高いですねー（SORACOMさんブースの人だかりも凄かった。。。

ブースに立ち寄られた方といろいろお話もしましたが、関西はまだまだオンプレで運用されている方も多く、これからクラウドに移行していかないといけないなーという意識を持っている方もけっこう多かったです。

でもそこって人や時間のコストがかかるから足踏みしちゃうとこなんですよね・・・

具体的にAWSのサービスを使ったらどういうことが出来るのか、どのプロダクトを使うことで自社のシステムと同じような環境を実現出来るのか、そのあたりを模索されている方が多かった印象です。

でもひとつひとつの製品について教えてもらえるセミナーとか勉強会ってなかなかないので、ほんとのほんとの基本の部分や、クラウドを知るきっかけになるような勉強会が開催出来たら、もう少し関西でのクラウド化の敷居も下がるのかもなーと思いました。

夜はナイトイベント＠JAWS-UG大阪

前半：パネルディスカッション

本編終了後はJAWS-UG大阪のイベントが開催されました。

前半は各コミュニティの代表が一堂に会するパネルディスカッション！

パネルディスカッションでは弊社西島がJAWS-UG代表として、金春がkintone代表として登壇！

（わたしはなんちゃってプレス風にもなれない程度の撮影班）

西島はイベント開催5分前に登壇が決まったようで・・・いやー圧力こわいこわい。。。

コミュニティの実状から方向性まで、パネリスト達が赤裸々に語り尽くした1時間！

西島は開催5分前に登壇が決まったにもかかわらず、安定の飲みっぷり＆語りっぷり( ∩ˇωˇ∩)

金春はこの時にはわりとできあがっていたようです。。。

コミュニティを知らない人達にも入ってもらいやすい雰囲気づくりを改めて考えるきっかけになった時間でした（｡╹ω╹｡）

後半：LT大会

後半は華麗なるLTer達のLT大会です！

去年は関西パイ姫という名誉ある称号を頂いたのですが、今回は残念ながらパイはなかったため、ドラ娘がエアパイっぽく頑張りました！（意味がry
