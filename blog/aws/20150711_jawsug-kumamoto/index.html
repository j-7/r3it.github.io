第5回JAWS-UGくまもと　クラウド勉強会に参加しました

Posted on 2015-07-11 17:30

Koichiro Nishijima

ノイズキャンセリングヘッドフォンの電池交換を忘れ、飛行機が滑走路から飛び立つ前に電池切れになって途方に暮れた西島です、こんにちは。

2015年7月11日の土曜日に、JAWS-UGくまもとに参加しまして、恐れ多くも基調講演としてセッションを持たせていただいたのでその報告です。

熊本といえば！

熊本といえば、そう、アレです。凛々しいサムライ姿のAWSサムライ2014が普段被ってるアレです。あれしか知りませんでしたゴメンナサイ。そこら中にあれがいました（笑）。

実は(?)人生初上陸の熊本でした。この登壇が決まってから人々は口々に「熊本はいいよ〜」と教えてくれたので、みんな行ったことあるんだなスゲェと思いつつ、「へ〜そうなんですね！」と期待に満ち満ちた笑顔を返していました。なるほど熊本城、凄かったです。

JAWS-UGくまもとについて

久しぶりの開催となったとのことですが、実は設立は2011年10月の古株支部であるくまもと支部、今日は第5回目の勉強会となりました。

午後からの大雨にもかかわらず、30名弱の参加者の皆様にお集まりいただきました。今回初参加という方が半分くらい居らっしゃいまして、久しぶりの開催ゆえのなかなか珍しい現象が起こったのかな？と思いました。

セッション内容

基調セミナーと題されて恐縮ですが、お話させていただいた内容は、割とシンプルな「初めてのクラウド AWS編」でした。資料はこちらです。テクニカルな話は極力無しに、クラウドそしてAWSの魅力を伝えようと頑張ってみましたが、いかがでしたでしょうか。ご意見ご感想など、お気軽にお送りいただけますと幸いです。

初めてのクラウド AWS編 from Koichiro Nishijima

ライトニングトークもしました！

タイトニングトークは元々2名予定だったのですが、一人二人と飛び込みが増えていった＆場を盛り上げるために（飛び込みウェルカムということだったので！）、昔作った資料をベースに「コミュニティに参加しようぜ！」というようなLTをしました。ハンズラボさま提供のドラがはるばる熊本の地まで空輸されてきたので、しっかり鳴らされておきました！

後半はハンズオン＋座学のダブルトラック

ハンズオントラックは、「Windowsクラサバ環境構築ハンズオン」と題してWorkSpacesがクライアント、サーバはWindowsサーバで！という環境を作る予定だったのですが、、、WorkSpacesの作成などなどで待ち時間が多数発生する事情により「これ今日は終わらんわ〜」という状況に陥ってしまいました(^_^;)

資料は共有されているので、あとは自分で頑張りましょう！ということで終了しました。

座学トラックの方は多田さんの「みんながクラウドつかうの応援するモン！」、吉田さんの「AWSへのWindows Server 2003の移行、そして今後インフラとどう向き合うべきか」、最後に地元熊本の松岡さんの「(県内事例紹介)新規事業開発におけるAWSのパワー」と、どのセッションも盛り上がっていました。

まとめ

新しい方が参加してくださった今回の勉強会、これにひき続いて細くなが～く継続していけるといいですね！微力ながら、支援させて頂きたいと思いますので、よろしくお願いします！

【コミュニティにフルコミット】を宣言するアールスリーでは、熊本に限らず呼ばれればどこにでも飛んでいって喋りますので、ぜひお声がけください！
