AWS Cloud Roadshow 2014に出展します

Posted on 2014-11-17

Toshiyuki Konparu

AWS Cloud Roadshow2014

AWS Cloud Roadshow2014とは、AWSが実施する展示会です。 全国４都市で実施されており、12/2の大阪が最後となります。

弊社も出展します。

詳細は、こちらにあります。

弊社の出展内容

今回は、”「AWS x kintone」でいままでにないシステム開発を!” として、cybozu社のkintoneとAWSを掛け合わせるソリューションについてご紹介いたします。 こちらの記事でも触れましたが、これからは複数のクラウドサービスを組みあせて使う形の「ハイブリッドクラウド」が盛り上がってくると考えています。

また、これらのクラウドの組合せをオープンソース・ソフトウェアを駆使した形で、できるだけスピーディにかつ安価にご提供できるようにしていきたいと考えております。

来場された皆様には、日頃のシステム開発でのお悩み事についてお聞かせ願えれば幸いです。 そのお悩みをハイブリッドクラウドで解決するべく弊社とともに解決していきましょう。

当日にはJAWS-UG Kansaiもあります

AWS Cloud Roadshowの後には同会場にて「JAWS-UG Knasai」も実施されます。 今回は、「AWS ゆく年、くる年」として今年のAWSを振り返り、来年の展望をみなさんでお話します。

AWSには非常に強力なユーザグループがあり、エンジニアの方だけではなくAWSに興味のある方々が全国でたくさん参加しています。 有志の集まったユーザグループですので、非常に和やかな雰囲気の中でAWSについて最新事例や他のユーザが何を思ってAWSを使っているかなどを知るいい機会になっています。

こちらのご参加もお願いいたします。
