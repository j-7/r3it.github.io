JAWS-UG沖縄 Cloud on the BEACH 2016 参加レポート

Posted on 2016-05-02

Koichiro Nishijima

ゴールデンウィーク中日の沖縄よりこんにちは、西島です。今年もこの季節がやってまいりました、JAWS-UG沖縄のCloud on the BEACH 2016の様子をお届けします。

Twitterタイムラインのまとめはこちらです。

※写真提供はGW帰省中のJAWS-UG沖縄公式カメラマン 栄野川さん(@enkw_)と、スマホ写真は西島です。

Cloud on the BEACH とは？

2012年の初回からはや5回目になります。毎年ゴールデンウィーク前に開催されている、JAWS-UG沖縄主催の最大のイベントでありお祭りです。朝から勉強会を開き、夕方からはビーチパーティをしよう、というのが企画の趣旨です。

GWに絡めた開催時期というのは、県外からのゲストを呼びやすい、そしてご家族で来ていただいて、そのまま沖縄を満喫して欲しい！というコアメンバーそして協力して下さる皆様の願いが込められております。ですが、この時期は毎年梅雨が始まるかどうかの瀬戸際なのです。毎年の神事である晴天祈願をしているのですが、いつもイベント直前の一週間は胃が痛い思いをしております・・・。

ところが今年は、このように素晴らしい天気に恵まれました！日差しは暑く夏っぽく、日陰に入れば涼しく、という絶好のビーチパーティ日和となりました。お楽しみ画像は後半に・・・。

トラック概要

今年も昨年同様未経験者向けと経験者向けのダブルトラックで開催となりました。未経験者向けトラックでは主に「クラウドとはなんぞや？」というところからの座学セッションが並び、経験者向けトラックでは突っ込んだ内容のセッションや、実際に手を動かすハンズオントラックが並びます。

今年も西島は未経験者向けトラックの方をファシリテーションしていましたので、そちらの様子を主に写真と資料でご紹介します！

初心者・未経験者向けゼネラルトラック

オープニングで軽くアンケートを取ったところ、半分程度が初めてのJAWS-UGへの参加者！そして学生さんもかなり多くいらっしゃったのが印象的でした。

ご参加いただきありがとうございます！今日がクラウドジャーニーの始まりです！！

「初めてのクラウド　AWSのご紹介」と「ITインフラを進化させるクラウドジャーニー」

朝イチと午後のセッションはAWS小島さんより「初めてのクラウド　AWSのご紹介」と「ITインフラを進化させるクラウドジャーニー」の2本立てです。

AWSの成り立ちが根本的に他のクラウドベンダーと異なること、そしてクラウドジャーニーの長い旅の途上にいるIT業界の事例を多数ご紹介いただきました。特にこのクラウドジャーニーの資料は、社内の抵抗勢力を説得できる非常に有用なマテリアルとなると思いますので、ぜひ皆様にご活用いただけると嬉しいです！

Cloud on the_beach_aws入門_公開 from Hideki Ojima

Cloud on the_beach_cloud_journey_公開 from Hideki Ojima

コスプレエンジニアがゆるゆるAWS全サービス紹介してみる

続いて弊社の池上がAWSの全サービス紹介にチャレンジしました。増え続けるAWSのサービスを1セッションで全て説明するというこのJAWS-UG沖縄恒例の企画、既に一昨年のサーバーワークス桶谷さんに無茶振りした段階でかなり時間的にキツキツなところを、自己紹介で時間を使う池上で大丈夫なのか・・・？と一抹の不安はありましたが、やはり全く時間が足りていませんでした（涙）。

「こういうものあるんだ〜」というインデックスとして聞いていただいて、詳しくは資料、そしてAWS公式サイトをご覧いただけますと幸いです。

20160429 JAWS-UG沖縄 Cloud on the BEACH 2016 AWS全サービス紹介 from Midori Ikegami

実際AWS入れてみたお話しをします

最後のセッションは、琉球インタラクティブ株式会社の照屋さんより「実際AWS入れてみたお話しをします」です。

マイクを使わず漢らしいトークで、実際にAWSを入れてみたらこうなったという話を、事例を交えてご紹介いただきました。非常に身近な事例ばかりで、ゼネラルトラックの皆さんにも参考になったのではないかと思います！

Cloud on the BEACH 2016 - 会社にAWSを導入した話 from Naoto Teruya

経験者向けトラックの様子も少しだけ

全く様子を見る時間がなかったので、写真を少しだけ貼っておきます。

座学セッションはカメラマンに登壇に大活躍のクラウドパック栄野川さんの「AWS構築のハマりポイントを詳しく解説」や、株式会社レキサス 与儀さんの「AWS WAFを用いたセキュリティ対策」、同じく株式会社レキサス 鈴木さんの「ネットゲーム開発者がサーバレスしてみた」などなど、流石にコアな経験者向けのコンテンツが熱く展開された模様です。

午後からは 「大革命！！kintoneとAWSで超ファストシステムを作ろう」というタイトルでサイボウズ株式会社の後迫さん・北川さん・kintone エバンジェリスト村濱さんのハンズオンセッション、モビンギの堀内さんよりモビンギハンズオンがありました！

ハンズオンは、みなさん「結構ついていくのが大変だったけど、面白かった！」と満足していたようですよ！

こちらに貼りきれなかった資料や写真は、JAWS-UG沖縄のFacebookグループにて共有していますので、当日参加された方はぜひご確認ください。

そしてビーチパーティへ！

ガッツリ勉強したあとは、ビーチパーティ開始です！

過去最大の参加者の皆様が、社会人・学生・県内・県外の人々が入り混じって、様々な話題で参加者皆様が盛り上がっていましたよ！

また、毎年鉄板奉行をお願いしている福嶺さんとサポートしてくれる上原さん、本当にありがとうございます！「参加者はビーチに行って飲んで食べるだけ」が実現できるのは、彼らのご協力があるからこそです。来年はもう少しお手伝い支援部隊が増やさないと、ですね。。。

でも沖縄の皆さんは流石手慣れたものです。鉄板掃除からビールサーバー「インスタンス」の交換、勿論ゴミ収集までテキパキと協力してくださるので、運営側としては大変ありがたいです。

また来年のいい天気でビーチパーティが出来るといいですね！

そして沖縄の熱い夜はなかなか終わりませんでした・・・。クラウドジャーニーはまさに始まったばかりなのです。

Still Day One!
