kintone で作ったアプリを HTML に埋め込む js2kintone のご紹介

Posted on 2015-07-06 18:30

Koichiro Nishijima

西島です、こんにちは。

今日は少し出遅れましたが AWS Lambda available in Tokyo! を記念しまして、先日オープンソースで公開しました js2kintone のご紹介をしたいと思います。

その前に、kintone とは？という方がいらっしゃいましたら、こちらの弊社ブログをご確認ください。

js2kintone とは？

js2kintone は kintone で作成したアプリを HTML ファイルに埋め込むことが出来る JavaScript ライブラリです。このライブラリを使うことで、kintone で作ったアプリをあなたのWebサイトにそのまま設置することが出来ます。

システム概要

システム概要は、以下のようになっています。

これ、いまご覧の弊社Webサイトが、この構成になっています。技術要素をかいつまんで説明しますと、

  • HUGO でHTMLファイル生成
  • Amazon S3 にてWebホスティング
  • Amazon CloudFront でコンテンツ配信 + Server Name Indication（SNI）独自 SSL
  • 問い合わせフォームでブラウザが直接 AWS Lambda を呼び出し、内容は最終的に kintone のアプリに格納

という流れです。

ブラウザから直接 Lambda を呼び出す例は、意外とまだ少ないようですので、参考にして頂ければと思います。

格安でのキャンペーン/コーポレートサイトのホスティングに最適

「月額10円〜レンタルサーバを始める方法」を活用する際にいつもお困りネタの「お問い合わせフォームどうするの？」という問題に対して js2kintone を利用すると、ほぼ無料でお問い合わせフォーム付きのコーポレートサイトを作ることが出来ます（勿論、若干のAWS費用と、そもそもの kintone の費用は掛かりますよ！）。

以下いくつかのFAQにお応えしていきます。

古いブラウザはどうするの？

いいところに気づきましたね。 AWS SDK for JavaScript in the Browser の Browser Support はこちらにあるとおりです。いわゆるモダンブラウザ、ですね。

js2kintone では、モダンではないブラウザに対しては、以下の様なアプローチで対応しています。

  • es5-shim.js を使って、スクリプト読み込み時にエラーを出さない
  • ブラウザを判別して、ごめんなさいメッセージを出す。

es5-shim.jsで AWS SDK が動くかどうかまでは流石に調べる気にならず、戦略的撤退ということで、お願いします・・・。

設置方法がなに言ってるかわからない

js2kintone のページからダウンロードや設置方法にリンクが張られてるのですが、うーん、難しくないものの確かに少し面倒かもしれないですね。

もし js2kintone を使ってみたいんだけどよく分からん、という方がいらっしゃいましたら、有料設置サービスも行っておりますので、ぜひお問い合わせください。

まとめ

AWS Lambda available in Tokyo! を記念しているのに、 js2kintone がオレゴンをオススメしているのは仕様です。探さないでください(^_^;
