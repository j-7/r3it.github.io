2016年5月 kintone API アップデートをチェック！

Posted on 2016-03-29

Koji Asaga

2016年5月の定期メンテンスで実施される kintone API のアップデート情報が公開されましたので、早速ですが内容をチェックしてみました。

2016/05/08 の定期メンテナンスにおけるkintone API、User API更新情報

注目の追加機能をピックアップ

今回は割りと地味目？なアップデートに見えますが、その中でも注目すべき機能追加がありましたので、ピックアップしてみたいと思います。

コメント関連 API

アプリのレコードコメントに関する API として、以下が追加されます。

  • レコードのコメント一括取得
  • レコードのコメント追加
  • レコードのコメント削除

kintone はコミュニケーションプラットフォームと言いながらも、今までコミュニケーション系の API が存在していなかったので、待望の機能追加といったところです。

例えば、専用のモバイルアプリから kintone にコメントを書き込むなど、これまでとは違った使い方ができるようになりそうです。

今回はレコードのコメントだけですが、スペースのスレッドや People にも拡張される事を期待したいですね。

kintone プロキシを使った添付ファイルのアップロードの対応

kintone の画面から外部の API を実行できる kintone.proxy が、添付ファイルのアップロードに対応します。

これにより kintone アプリに保存された添付ファイルを他のシステムに連携する事が可能となります。

例えば、kintone に画像を保存しておき、kintone で承認が行われたらウェブサイトで公開するといった事が可能になります。

まとめ

ピックアップした機能以外にも、かゆいところに手が届くようになる嬉しい機能が追加されます。

今まで諦めていた事も実現できるようになり、kintone を利用できる幅が益々広がりそうですね。

kintone についてお困りの事があれば、お気軽にご相談ください。
