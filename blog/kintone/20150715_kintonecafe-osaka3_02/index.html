kintone Café 大阪 Vol.3で実施したカスタマイズハンズオンの概要について(後編)

Posted on 2015-07-24

Ikegami MIdori

2015年7月15日水曜日に、kintone Café 大阪 Vol.3でJavaScriptを利用したカスタマイズハンズオンを行いました。

前回のブログでは、kintoneにおけるJavaScriptのカスタマイズルールについて少し説明させて頂きましたが、今回は実際にハンズオンで利用したサンプルコードについてもう少し理解を深めていきたいと思います(‘ω’*)

用意したアプリについて

前半もお伝えした通り、こちらで予め作成させて頂いたJavaScriptのサンプルコード。ひとつずつどんな風に記述されているか見ていきましょう(⑅っˇωˇ )っ

バリデーション

そもそもバリデーションとは何でしょう？

規定された要求事項がその記述された使用に対して適切であることを検証すること。

…とかググると小難しいことが書かれていたりしますが、要は入力された値の○×チェックです。

入力された値が条件に合ってなければ、エラーを返すようなしくみになります。

記述方法

では実際にどんな書き方をするのか見ていきましょう！

ハンズオンの中では、郵便番号、TEL/FAX番号、メールアドレスの半角/全角チェックを入れていました。

半角/全角チェックとはいいますが、郵便番号、TEL/FAX番号に関しては、半角数字で決まった桁数入っているかをチェックしているだけです。

// 郵便番号の書き方ルール(ハイフン必須。7桁の半角数字)
var zip_pattern = /^\d{3}-\d{4}$/;

// TEL番号, FAX番号の書き方ルール(ハイフン必須。10桁または 11桁の半角数字)
var telfax_pattern = /^(0\d{1,4}-\d{1,4}-\d{4})$/;

メールアドレスに関してのみ、半角英数字と@を利用するので、少しルールが複雑になっています。

このルールでは、半角英数字と、@が必ず入っていないとエラーになります。 

// メールアドレスの書き方ルール (一般的な書き方)
var mailpattern = /^([a-zA-Z0-9])+([a-zA-Z0-9.-])*@([a-zA-Z0-9-])+([a-zA-Z0-9.-]+)+$/;

入力された値が、上記で定めたルールにマッチしているかを下記のように書いて確認しています。

　※ここではTEL番号を例に挙げていますが、郵便番号、FAX番号、メールアドレスも書き方は一緒です。

if (tel_value.length > 0){
    // TEL番号のルールにマッチしているかをチェックしています。
    if (!(tel_value.match(telfax_pattern))){
        // マッチしない場合は、エラーの内容を記載します。
        record.TEL.error = '10桁 または 11桁の半角数字（ハイフン有り）で入力して下さい。';
    }
}

バリデーションという言葉は聞き慣れない方もいるかもしれませんが、コードの書き方としては、郵便番号、TEL/FAX番号、メールアドレスそれぞれに対して、上記のようなチェックを繰り返しているだけです。

住所から地図を表示する

2つ目は、住所フィールドに入力した情報から緯度経度情報を取得し、赤枠で示したスペースフィールド（Maps）に地図を表示するというものです。

以下の記事と同じような考え方になります。

https://cybozudev.zendesk.com/hc/ja/articles/202640950

これは解説していると少し長くなってしまうので省略しますが、この「要素」を使った書き方はわりとkintoneでもよく使うので、押さえておくと良いかもしれませんね！

  • 要素（エレメント） = 見出し、段落などのパーツ。
  • div要素 = スタイルシートの適用や、JavaScriptで操作する時によく利用される

記述方法

// 地図を表示するdiv要素を作成します。
var elMap = document.createElement('div');
elMap.setAttribute('id', 'map');
elMap.setAttribute('name', 'map');

// 地図表示用に追加した空白スペース（Maps）にelMapで設定した要素を追加します。
var elMapContainer = kintone.app.record.getSpaceElement('Maps');

// 地図用div要素をこのMaps要素の子要素として追加します。
elMapContainer.appendChild(elMap);
