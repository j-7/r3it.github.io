Cloud Days Tokyo 2016 に出展してきました！！

Posted on 2016-03-14 15:00

Ikegami MIdori

デブサミ2016の登壇報告以降ブログ更新が滞っておりすみません。

2016年3月10日・11日の2日間、Cloud Days Tokyoに出展してまいりましたのでそのご報告をさせて頂きます！

Cloud Days って？

日経BP社主催の大規模なITイベントです。今年2016年はクラウド、ビッグデータ、モバイル＆ウエアラブル、セキュリティ＆ガバナンスそしてIoTにフォーカスし、東京・大阪、九州・名古屋・札幌の全国5ヶ所で開催されます。

（近くにテスラが停めてありました。クラウドかIoT!?）

Cloud Days Tokyo は初出展！

昨年は大阪、名古屋、九州の3ヶ所で出展させて頂きましたが、今年初の Cloud Daysはなんと東京から！

今回は昨年の名古屋の時のように、サイボウズさんのパートナーブースで出展させて頂きました。

普段kintone界隈でお世話になっているジョイゾーさんやM-SOLUTIONSさんなども出展されていて、華やかな雰囲気でしたよ！

M-SOLUTIONSさんのブースではお馴染みのペッパーくんがお出迎え！（写真撮り忘れました・・・）

サイボウズさんのブースではミニセミナーも！

今回はサイボウズさんと各パートナー様のペアでセッションが展開されるスタイルでした。

弊社からは金春が「小さく・はやく」作ることの大切さを訴える、ハイスピードSIについて登壇させていただきました。

（少し引いてみると、この人だかり！）

kintoneとAWSの連携ソリューションの展示

ブースの方では、kintoneとAWSの連携ソリューションを展示しておりました。

kintoneと外部サービスとのさまざまな連携パターンを、皆様に知って頂くことが出来たかなと思っております(๑•̀ㅂ•́)و✧

Cloud Days 2016 東京のまとめ

今回のイベントの参加者で、弊社ブースにお立ち寄り頂いた方にお話をお聞きすると、kintoneの名前自体をご存知ない方、また名前は知っていても使ったことがない方が多かったように思います。

サイボウズさんの製品ということをご説明させて頂いて初めて知った！という方も。

ただそういった方にデモをお見せして、こういった使い方が出来るんだよということをお伝えすると、かなり高い確率で興味関心を持って頂けました！

今週末3月17日(木曜日)・18日(金曜日)はCloud Days 大阪に出展します！

大阪の皆様！是非会場まで足を運んで下さいね(๑´ㅂ`๑)

大阪事前登録はこちらです！ http://expo.nikkeibp.co.jp/cloud/2016spring/osaka/regist.html
