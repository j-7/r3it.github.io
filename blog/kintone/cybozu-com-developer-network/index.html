cybozu.com developer networkについて

Posted on 2014-12-01

Toshiyuki Konparu

こんにちは、金春です。

kintoneを始めたいと思っている方、「お試し期間1ヶ月じゃ試しきれない」という方、 意外と知らない方も多いかもしれませんので、cybozu.com developer networkをご紹介します

cybozu.com developer networkとは？

cybozu.com developer network（長いの以降、CDNと略します）は、cybozu.comの開発者が集まるコミュニティサイトです。 kintoneなどのカスタマイズ方法について、フォーラムで質問したりすることができます。

わからないことなどあれば、ここで聞いていただくと、kintoneエバンジェリストの方々などから回答もらえるかもしれません。

開発者ライセンスを取得しよう

CDNのすごいのは、開発者ライセンスがもらえるということです。 開発者ライセンスをもらうと、1年間kintoneのスタンダードコースが使い放題です。

ひとまず、CDNのアカウントを取得しましょう。「メンバーになる」をクリックしてください。 Facebookのアカウントがあれば、すぐにメンバー登録できます。

メンバー登録ができたら、登録したアカウントでログインしてください。

次に赤枠のところをクリック！

そして、画面の下の方にある申込みフォームを埋めます。

「確認する」をクリックして、送信すれば、２〜３日でアカウントがもらえます！ これでkintone触り放題です！

まとめ

CDNで開発者ライセンスをもらえばじっくりとkintoneを触ることができますので、 是非お試しください。

通常のライセンスとは異なるいくつかの制約がありますが、開発上困ることはないと思います。
