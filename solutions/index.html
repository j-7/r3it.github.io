ハイスピードSIとは？

R3 instituteは、設立当初からのテーマとしての、

従来のシステム開発の非効率な部分を何とかできないものか

を考え続けてきました。

営業にいく先々で、「システムというのは、なぜこんなに費用と時間がかかってしまうのか」というご相談をうけ、そこに真剣に向かいあってきました。

本来ITは、業務を、そして経営を助ける手段でなければならないはずが、 いつのまにか、「あのシステムを変えるのに費用がかかりすぎるから、業務を変えることが出来ない」 といった、経営にとってお荷物的な存在となってしまっている話も、幾度となく聞いてきました。

このままでは、日本企業の国際競争力にすら響いてしまうのではないか、そんな危機感すらあります。

２０１５年、、、世間では様々なクラウド技術、クラウドサービスがかなり台頭してきました。

一つ一つは単なる「部分」のサービスですが、「これらのクラウドをどううまく使いこなすか？」と いう、我々が得意とする　SI　としてのコーディネート力で考えたときに、 従来からのSIとしての課題にもフィットすると思えたのです。

このコンセプトこそ

「ハイスピードSI」

構成要素として、我々がもっとも最適だと思っているのは以下の３つです。

Amazon Web Services (AWS)

  • インフラとしては、クイック、柔軟、多機能かつ低コストなAWSを用いる。
  • 従来の様々な面倒なこと、コストがかかっていたことから解放するとともに、従来だとあり得なかったインフラサービスを提供する。
  • AWS前提にしたときには、アプリの作り方も変えられ、新たな世界感が出来てきます。

サイボウズ kintone

  • アプリ部分は、ノンプログラミングで出来る限り行う。
  • kintoneであれば、ミドルLayer、インフラLayerも気にしなくて良くなるため、上流～コーディング～下流の各工程で、かなりの効率化が図れます。

R3のシステム開発サービス

  • 設立以来、様々なお客様に評価されてきた、企画力、設計・開発力、柔軟性、コスト、スピード。
  • AWSやkintoneを単なるサービスとして捉えるのではなく、従来のスクラッチ開発とのハイブリッドで考えることにより、クラウドの力を十二分に発揮させることができます。

この世界観をたくさんの方に知っていただきたいと思っています。
