AWSについて

Amazon Web Services (AWS)とは

AWSとはAmazon Web Services（アマゾンウェブサービス‎）の略称です。世界最大のパブリッククラウドサービス事業者であり、同社は世界最大級のオンライン小売事業者であるAmazon.comのインフラを支えています。

弊社アールスリーインスティテュートはAWSコンサルティングパートナーに認定されており、過去の実績とともに培ったノウハウで、お客様のAWS導入を支援します。

R3 instituteのAWS関連サービス

これまでエンタープライズ向けに、様々な業界の様々なシステム開発をしてきた弊社であれば、安全性、費用面、柔軟性など、どこに重点を置くかをお客様と共有することが可能です。その結果として、決まり切った構成ではないご提案やオプション機能のご提案も得意とします。

AWSコンサルティングサービス

弊社であれば、AWSに特化せず、様々な他社サービスも視野に入れたうえでのご提案が可能です。
お客様のニーズをヒアリングさせて頂いたうえで、AWSを使った最適な構成を設計し、費用面とともにご提案いたします。

AWS構築サービス

様々なお客様のニーズに従ったAWSの構築および動作テストを行います。（以下は例）
  • 業務系システム（自動復旧機能あり／なし）
  • 一般消費者向けWebシステム（自動サーバ追加あり／なし）
  • 大量トランザクションデータを扱うシステム
  • ファイルサーバの代替
  • コスト節約型のサーバ稼働（自動ON/OFF機能）AWS構築サービス
（オプション）AWSの上にのせるアプリケーションの開発およびセットアップも可能です。
（他社アプリケーションのセットアップや保守引継も対応可）

AWS運用保守サービス

  • AWS環境のサーバ運用維持のための監視、保守対応を行います。
  • AWSへの支払い代行（ドル・クレジット・変動料金　→　円・請求書・固定料金）

AWSを利用した構築事例

事例：株式会社夢ふぉと様

http://www.yumephoto.com/

解決したかった課題

株式会社夢ふぉと様では、弊社が開発した顧客管理システムをオフィス内で設置したサーバで運用していました。しかし、オフィス内での運用では、以下のような問題がありました。

  • サーバの老朽化に伴い、サーバのリプレイス作業が必要となる
  • サーバのメンテナンスがリモートで実施できず、訪問しての作業となるため対応に時間がかかる

このような問題を解消するため、弊社にて顧客管理システムをAWS上に移行しました。

事例：株式会社トムス・エンタテインメント様

http://ani-me.jp/

解決したかった課題

株式会社トムス・エンタテインメント様では、視聴者が見たアニメを記録管理するサービスとしてアニme http://ani-me.jp/ を提供されています。 このサービスでは、企画当初から以下のような課題がありました。

  • 新規サービスであるためどの程度のアクセスがあるかが予想できない
  • 人気アニメの放送中や直後はアクセスが集中する可能性がある
  • サービスを徐々に成長させていくためにも初期投資を抑える必要がある

このような問題を解決するため、当初よりAWS上にシステムを構築することとなりました。
