アールスリー、gusukuアンバサダー・プログラムを開始

平素は弊社の gusuku（グスク） をご利用頂き、ありがとうございます。

ハイスピードSIを提唱するシステムによる問題解決会社であるアールスリーインスティテュート（本社：大阪市中央区、代表：加藤忠智）は、本日アールスリー、gusukuアンバサダー・プログラムを開始いたしました。

gusukuアンバサダー・プログラムはgusukuを応援してくださる方を「アンバサダー」として任命させて頂くプログラムです。

詳しい内容は、gusuku（グスク）公式サイト https://gusuku.io/ambassador/ にてご確認ください。
