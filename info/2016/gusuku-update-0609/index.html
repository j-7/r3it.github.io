アールスリー、kintoneのデータバックアップ機能をgusukuにて提供開始

Posted on 2016-06-09

https://gusuku.io/

平素は弊社の gusuku（グスク） をご利用頂き、ありがとうございます。

ハイスピードSIを提唱するシステムによる問題解決会社であるアールスリーインスティテュート（本社：大阪市中央区、代表：加藤忠智）は、本日kintoneのレコードデータと添付ファイルをクラウド上にバックアップするバックアップオプションをリリースいたしました。

gusuku上の簡単な操作でkintone上のレコードデータ、添付ファイルをアプリ単位でバックアップ・リストアすることが可能となります。CSVファイル形式ではリストアできない項目もきちんとバックアップ・リストアできるので、データ消失への対策となるのはもちろん、アプリケーション開発時のマスターデータ管理をバージョン管理と同等の手軽さで行うことが可能です。

バックアップオプションは、フリープランも含めすべてのお客様でご契約いただけるオプションとなります。また、1組織あたり20MBまで無料でご利用いただけます。

お申込は公式サイトの「オプション申込」にて承ります。

「バックアップオプション」につきましての詳細は gusuku docs のオプションページをご覧ください。

gusuku（グスク）について

「gusuku（グスク）」は kintone 上のアプリケーションのバージョン管理、配布管理を行えるほか、gusuku APIを利用した kintone プラグインを利用することによって、kintone 自体をさらに便利にご活用頂けるプラットフォームとなっております。

無料のフリープランもございますので、まずお試しいただくことが可能です。すべての kintone アプリ開発者の皆様にご利用いただくことを願っております。

こちらの公式サイトよりフリープランの「申込」をクリックしてサインインしてください。

gusuku（グスク）公式サイト： https://gusuku.io/
