Talend Open Studio パートナーに登録されました

Posted on 2015-08-06

弊社アールスリーインスティテュートはこの度、 Talend Open Studio パートナーに登録されました。

Talend Open Studio パートナー（TOSP）とは

Talend Open Studio (TOS)のトレーニング、サポート、コネクタやモジュールの受託開発などを提供するパートナーです。公式サイトはこちらのURLになります。

https://talend.jp/tosp

弊社アールスリーインスティテュートでは、以下のコンポーネント・サービスを提供しています。

  • Talend Open Studio 向けの サイボウズ kintone と接続するためのオープンソース コンポーネント”kintone Connector for talend” の提供
  • 上記コンポーネントを利用した kintone と既存システムのデータ連携システムの構築
  • Talend Open Studio を利用した基幹システムと Amazon Web Services (AWS) との間でのシステム・データ連携

システム概要イメージ

Talend Open Studio と kintone を利用したデータ連携システムの概要は、上記のとおりです。AWS上に構築した基幹システムと、サイボウズ側ネットワークにある kintone アプリ群を、 Talend でデータ連携するイメージです。

システム間のデータ連携にご興味があるお客様は、是非お問い合わせください。
