JAWS-UG Kansai 特別編に弊社金春、大澤が登壇いたします

Posted on 2015-02-02

JAWS-UGとは？

JAWS-UG（Japan AWS User Group）は、AWSを利用するユーザの集まりで、全国に50の支部を持つ世界でも有数のAWSのユーザグループです。

JAWS-UG Kansai 特別編 「AWSを使い倒せ。AWSのフルマネージドサービス活用によるネイティブクラウドシステムへの誘い」

2015年2月7日(土)に開催される、 JAWS-UG Kansai 特別編 「AWSを使い倒せ。AWSのフルマネージドサービス活用によるネイティブクラウドシステムへの誘い」 に弊社金春、大澤がスピーカーとして登壇します。

この勉強会では、AWSのフルマネージドサービス（運用はAWSが実施しお客様はサービスを利用するだけで使えるサービス群）についての紹介や、ビッグトラックでは、東急ハンズ様やあきんどスシロー様といったクラウド先進企業の方から、どのようにAWSを利用しているかのご公演もあります。

金春は、 15:00〜15:30 Amazon RDS for Aurora 17:30〜18:00 Amazon Kinesis

大澤は、 15:30〜16:00 Amazon SES

にて、各サービスのご紹介をいたします。
